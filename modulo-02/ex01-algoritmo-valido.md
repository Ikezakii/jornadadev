Exercício 1 — Isto é um algoritmo válido?
Para cada situação abaixo, indique se é um algoritmo válido. Se não for, explique por quê
(lembre dos três requisitos: finito, ordenado e sem ambiguidade).

a. “Some dois números.”
b. “Leia um número. Se for positivo, mostre ‘positivo’. Senão, mostre ‘negativo’.”
c. “Fique repetindo até acabar.”
d. “Pegue um ingrediente qualquer e cozinhe.”

Resolução:

a -> Algoritmo válido

b -> Algoritmo válido

c -> Algortimo inválido, não especifica o que deve ser repetido (Repetir palavras? Repetir Ações?) e nem quando acabar, esse "até acabar" pode ser varias coisas "Acabar a água", "Acabar o dia" etc

d -> algoritmo inválido, Instruções abstratas que não direcionam corretamente quem for executar a tarefa, "Um ingrediente qualquer", pode ser um tempero, uma carne, arroz, feijão, "e cozinhe", cozinhar qual receita, uma receita com o ingrediente selecionado? Uma receita especifica? Ou simplesmente "cozinhar" o ingrediente?
